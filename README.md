# undue-hardship.openbankruptcyproject.org

**The Undue Hardship Standard**

Reference page on the legal standard for discharging student loans in bankruptcy under 11 U.S.C. § 523(a)(8). Covers the Brunner test, the totality-of-circumstances alternative, and circuit-level variation.

Live site: <https://undue-hardship.openbankruptcyproject.org/>

Maintained by the [Open Bankruptcy Project](https://openbankruptcyproject.org/), a 501(c)(3) public charity (EIN 41-5159631).
